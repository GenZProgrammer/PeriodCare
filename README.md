# PeriodCare

Aplikasi Android untuk memprediksi siklus menstruasi, masa subur, ovulasi, dan estimasi Hari Perkiraan Lahir (HPL).

## Fitur

* Login dan Registrasi menggunakan SQLite
* Prediksi Haid Berikutnya
* Prediksi Masa Subur
* Prediksi Hari Ovulasi
* Estimasi HPL
* Kalender Interaktif
* Pengingat Notifikasi
* Voice Assistant (Text To Speech)
* Logout dan Multi Akun

## Teknologi

* Java
* Android Studio
* SQLite Database
* Material Calendar View
* SharedPreferences
* Text To Speech (TTS)
